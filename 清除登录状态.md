# 清除登录状态指南

## 方法1：通过浏览器开发者工具（推荐）

1. 打开应用：http://localhost:3000
2. 按 `F12` 打开开发者工具
3. 切换到 **Console** 标签
4. 输入以下命令并回车：
   ```javascript
   localStorage.clear()
   location.reload()
   ```
5. 页面会自动刷新并回到登录页面

## 方法2：通过浏览器设置

### Chrome/Edge
1. 打开 http://localhost:3000
2. 按 `F12` 打开开发者工具
3. 点击 **Application** 标签
4. 左侧找到 **Local Storage** → **http://localhost:3000**
5. 右键点击 **token** → **Delete**
6. 刷新页面（F5）

### Firefox
1. 打开 http://localhost:3000
2. 按 `F12` 打开开发者工具
3. 点击 **Storage** 标签
4. 左侧找到 **Local Storage** → **http://localhost:3000**
5. 找到 **token**，右键 → **Delete Item**
6. 刷新页面（F5）

### Safari
1. 打开 http://localhost:3000
2. 菜单栏：**开发** → **显示Web检查器**
3. 点击 **Storage** 标签
4. 左侧找到 **Local Storage** → **http://localhost:3000**
5. 删除 **token**
6. 刷新页面（Cmd+R）

## 方法3：清除所有浏览器数据

### Chrome/Edge
1. 设置 → 隐私和安全 → 清除浏览数据
2. 选择"Cookie及其他网站数据"
3. 时间范围选择"过去1小时"
4. 点击"清除数据"

### Firefox
1. 设置 → 隐私与安全 → Cookie和网站数据
2. 点击"清除数据"
3. 确认清除

## 为什么会这样？

因为应用使用 `localStorage` 保存登录 token：

```javascript
// 登录时保存
localStorage.setItem('token', 'your-token')

// 应用启动时检查
const token = localStorage.getItem('token')
if (token) {
  // 已登录，跳转到相应页面
}
```

由于你之前用手机登录过，token 还保存在浏览器中，所以应用认为你已登录。

## 如何避免这个问题？

### 开发时使用无痕模式
- Chrome/Edge: `Ctrl+Shift+N` (Windows) 或 `Cmd+Shift+N` (Mac)
- Firefox: `Ctrl+Shift+P` (Windows) 或 `Cmd+Shift+P` (Mac)
- Safari: `Cmd+Shift+N`

每次关闭无痕窗口，所有数据都会被清除。

### 使用不同的浏览器
- 开发时用 Chrome
- 测试时用 Firefox/Edge
- 移动端测试用手机浏览器

## 添加退出登录按钮

你也可以在个人资料页面点击 **Logout** 按钮退出登录。
